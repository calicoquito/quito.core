# Quito-Core
The core plone installation for quito

# Before you start 
Ensure you have [plonecli](https://github.com/plone/plonecli) installed.

# Getting started

```
git clone https://github.com/calicoquito/Quito-Core
cd Quito-Core/
git fetch
git checkout ploneclient
cd src/quito.core
plonecli build --clean
plonecli serve
```
